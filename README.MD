# Personal Finance Tracker Bot

A Telegram bot for tracking expenses with Google Sheets integration.

## Setup

1. Copy `.env.example` to `.env`
2. Fill in your environment variables
3. Deploy to Railway/Render/PythonAnywhere

## Environment Variables

- `BOT_TOKEN`: Telegram bot token
- `GOOGLE_CREDENTIALS_JSON`: Google service account credentials
- `SPREADSHEET_NAME`: Your Google Sheet name
